## ✨ Funcionalidades da Aplicação

- **🎨 Design Responsivo & Temas:** Interface moderna construída com Tailwind CSS, adaptável a dispositivos móveis e suporte a Dark/Light mode.
- **💬 Contato via WhatsApp Integrado:** Formulário interativo com validação no client-side e disparo formatado via API do WhatsApp.
- **🔔 Feedback Visual (Toast Notifications):** Notificações em tempo real para ações do usuário sem necessidade de modais intrusivos.
- **⚡ Performance & Acessibilidade:** Estrutura HTML5 semântica e carregamento otimizado de ativos visuais.
